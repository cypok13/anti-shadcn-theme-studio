## Архитектура токенов

ThemeTokens (src/lib/themes/types.ts) — TypeScript-интерфейс, управляется через Token Editor и пресеты. При смене пресета resolve.ts конвертирует camelCase поля в CSS custom properties и инжектирует их в iframe через postMessage. Результат виден в preview iframe как inline styles на :root.

globals.css — стили AppShell (основного приложения). :root блок содержит дефолтные значения токенов (Zinc Violet light). Эти значения применяются до загрузки любого пресета. Файл читается браузером напрямую — не через ThemeTokens.

Sidebar-токены (sidebarBackground, sidebarForeground и др.) существуют в ThemeTokens и пресетах, но отсутствуют в globals.css — они предназначены только для iframe preview, где ComponentGallery рендерит sidebar-компоненты с полным набором токенов.
